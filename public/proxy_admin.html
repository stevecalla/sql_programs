System reference

• What this is: a single reverse proxy on :8000 fronting every server_*.js at usat-api.kidderwise.org/<prefix>. Backends keep their own ports; the proxy strips the prefix before forwarding.
• URL pattern: usat-api.kidderwise.org/<prefix>/<path> → backend gets /<path>. e.g. /events/scheduled-events → 8005 /scheduled-events.
• Add / change a route: 1) edit proxy_routes.js   2) npm run pm2_reload_proxy   3) test the URL.
• Reload vs restart: pm2_reload_proxy = zero-downtime; restart_proxy = hard, brief blip. Deploy a backend with pm2 restart <name> — the proxy is untouched.
• Health: /api/health 200 = all up; 503 = one down (the proxy log names which). ECONNREFUSED = backend not running.
• Docs: plans_and_notes/PROXY_PLAN.md, PROXY_CUTOVER_CHECKLIST.md. Ops menu: npm run menu.
