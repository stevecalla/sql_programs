System reference

Program objectSalesforce sObject holding the Sanction ID via cfg_Id__c = BLANKVALUE(cfg_Legacy_Id__c, cfg_Autonumber_ID__c). Default Program. Change only if your org renamed/recustomized it (Settings → Advanced).
Live APIsSalesforce SOSL/SOQL (file intake) · Slack files.list (Slack intake) · /api/event (analytics ingest) · ngrok (optional public tunnel).
Database tablesrace_results_transform_events (usage) · race_results_transform_ask_log · race_results_transform_ask_corrections.
Key env varsRACE_RESULTS_ADMIN_USER/_PASS (admin) · RACE_RESULTS_CONVERTER_METRICS_USER/_PASS (app) · SF_PROD_*/SF_DEV_* · SLACK_BOT_TOKEN · SF_PROGRAM_OBJECT · NGROK_AUTHTOKEN.
Auth modelApp login → / + Salesforce/Slack/Folder intake. Admin login → /metrics + /admin. The .env account is the non-removable recovery; extra users live in the gitignored admin_overrides.json.
Slack setupBot scopes files:read, channels:read, channels:history, groups:read, groups:history, users:read; put the xoxb-… token in .env; /invite @bot in any channel and it appears in the picker.
