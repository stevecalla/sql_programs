# README — reporting app (plans & notes index)

The USAT Reporting app: a React SPA + Express host for USAT reports, behind the `:8000` proxy at
`/reporting`. First page: participation maps. Built to mirror `src/salesforce_merge`.

Start here:

- **PHASE_PLAN.md** — the phases (0–5), what each delivers, and acceptance criteria.
- **STATUS.md** — what's built so far and the exact next steps.
- **METRICS_AND_ADMIN_OVERLAP.md** — how auth/metrics/admin relate to the merge app (decision + options).
- **DEPLOY_AND_PROXY.md** — npm scripts, build, PM2, and the manual go-live (uncomment the proxy line).

Top-level app docs: `src/reporting/README.md` (run instructions), `notes.txt` (working notes),
`CLAUDE.md` (conventions for anyone extending it).
