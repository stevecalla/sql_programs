# Context file template (copy, rename, fill in, then drop into data/context/_global or a queue folder)

## Topic / question
A short heading describing the topic (e.g. "Coaching recertification timeline").

## Verified facts
- State the real, verified fact here (e.g. "Recertification review takes about 3 weeks from submission.").
- Include exact figures, dates, prices, and the correct contact email/URL.
- Only put facts you want the AI to state to members.

## Who/where to route
- If this should be escalated or routed, say to whom (team / email).

(Tip: keep each file focused. The assistant reads everything in the folder, capped ~20k chars total.)
